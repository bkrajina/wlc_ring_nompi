.. _analyzing:

Analyzing Simulation Output
###########################

Assuming you've used the :py:mod:`wlcsim` module for running your simulations,
whether they be parameter scans or simple replicas, there are numerous classes
and helper functions in the :py:mod:`wlcsim.data` for loading the output, and
functions in the module :py:mod:`wlcsim.analysis` for analyzing the data once
it's been loaded.
